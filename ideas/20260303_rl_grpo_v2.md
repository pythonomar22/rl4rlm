# 20260303: GRPO RL Training v2 (Hard Tasks)

## Hypothesis
GRPO RL training including 50K and 100K documents will improve accuracy on long-context tasks where SFT plateaued at 80%. Training on tasks the model actually fails at (rather than easy ones) will provide real advantage signal.

## Method
1. Starting checkpoint: SFT v2 (84% on NIAH-25)
2. GRPO: K=4 trajectories per prompt, group-relative advantages
3. Task pool: 50 NIAH tasks across ALL lengths (5K, 10K, 20K, 50K, 100K)
4. 40 steps × 2 prompts/step = 320 trajectories generated
5. Reward: 80% binary correctness + 20% format quality
6. LoRA lr=5e-6, AdamW
7. Key difference from v1: includes 50K-100K tasks where model fails ~20%

## Expected Outcome
- 90-95% on scaled NIAH (100 tasks)
- Specific improvement at 50K and 100K (80% → 90%+)
- Non-zero score on Multi-Needle NIAH and Document Classification

## What Would Disprove This
- ≤84% (no improvement over SFT)
- Improvement only on short docs (RL not learning long-context strategies)
- Position 0.25 still stuck at ~60% (structural limitation, not RL-solvable)
